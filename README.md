# Mostafa Eldiwany | Fullstack Engineer & Creative Developer

This is the source code for the personal portfolio of Mostafa Eldiwany, a Fullstack Engineer and Creative Developer specializing in building scalable web applications and intuitive AI experiences.

The portfolio is built with:
*   **Frontend:** React, TypeScript, Tailwind CSS
*   **Animation:** Framer Motion
*   **Design:** Futuristic, high-performance aesthetic

## Features

*   Interactive Hero Section
*   Dynamic Project Showcase
*   Speedcubing and Professional Stats
*   Custom Cursor and Background Effects

## Setup and Run Locally

**Prerequisites:** Node.js (v18+)

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```

The application will be available at `http://localhost:3000`.
